## Información del Proyecto

### Contexto Corporativo
**Lucky Shop** es un emprendimiento dedicado a la venta de accesorios presentados en "bolsitas de la suerte" tanto para dama como para caballero. Ante la necesidad de profesionalizar el negocio, hemos desarrollado un sistema integral que migra la gestión manual (basada en WhatsApp, TikTok y cuadernos) hacia una plataforma tecnológica unificada, permitiendo escalar el emprendimiento de forma organizada.

### Propuesta de Valor
Nuestra plataforma automatiza los procesos clave del negocio para eliminar la ineficiencia operativa mediante:
1.  **Gestión Centralizada:** Unificación de inventario, ventas y clientes en una sola base de datos en la nube.
2.  **Experiencia de Compra:** Aplicación móvil y web que permite a los clientes realizar compras, gestionar su perfil y ver el catálogo sin fricciones.
3.  **Inteligencia de Negocios:** Reportes automáticos de ganancias y segmentación de clientes frecuentes para la toma de decisiones.

### Objetivos Principales
* **Optimización Operativa:** Eliminar el registro manual y el desorden mediante un sistema automatizado.
* **Escalabilidad:** Preparar el negocio para la contratación de empleados y la expansión del catálogo (charms y bolsitas personalizadas).
* **Profesionalización:** Proyectar una imagen moderna y confiable para captar clientes en nuevos segmentos de mercado.

### Backend (Lógica y Base de Datos)
* **Lenguaje:** JavaScript (Lógica unificada en todo el stack).
* **Entorno de Ejecución:** Node.js (Servidor de alto rendimiento).
* **Framework:** Express (Manejo de rutas y APIs).
* **Base de Datos:**
    * **MongoDB Compass:** Interfaz gráfica para la gestión y consulta de datos.
    * **MongoDB Atlas:** Servidor en la nube conectado al gestor.

### DevOps y Entornos
* **Android Studio:** Entorno de compilación y emulación para la app móvil.
* **Vercel:** Despliegue y hosting de la aplicación frontend web.
* **Render:** Hosting para el servidor Backend y servicios en la nube.
